这片竹林被深厚的迷雾所笼罩，身处其中非迷路的人类会产生同样的景象在眼前不断延伸的错觉。排列并不规则的%深绿主色竹子%主色主色，平缓的地面起伏，没有任何能用以充当路标的标志物。
即使想要在其中笔直前进，不知不觉间又绕回了原点。这，就是被称作迷失竹林的迷惘地。
我的永远亭位于这片让人彷徨的竹林之中。尽管常年在此生活，已然不记得经历了几度四季更替，对于我而言，以某个事件为契机，时间的指针不过才刚刚开始摆动而已。
由于二、三年前地上的人类发动的袭击事件，我停止对永远亭施加永远的魔法。
#

若要问这是为什么，那是因为我看到了人类和妖怪相互协助、亲密无间的融洽模样，并对此感到极度羡慕。同时也觉得永远在躲避月之使者的追捕、战战兢兢地过着隐居生活的自己愚蠢至极。
所谓永远的魔法，是一种停止施加对象的历史行进，使其沾不上任何污秽、拒绝任何变化的魔法。
生物停止生长，食物永不腐烂，易碎品掉落依旧完好无损，覆水回收，这一切都是永远的魔法造就的奇迹。
#

我出于月之民的自觉，对地上世界的污秽感到恐慌，将这种魔法施加在了整栋建筑物上。
然而亲眼目睹地上之民的魅力后，我又主动解开了魔法。
其结果自然是永远亭被地上世界的污秽侵蚀。
食物不尽早吃掉马上便会腐烂，饲养的任何动物都有寿命这一概念，昂贵的古董壶必须小心谨慎地搬运。
取而代之的是，原本充满对来自月之都的使者的畏惧的隐居生活变得明朗而又丰富多彩。
#

而生活在失去了永远二字意义的永远亭中的我们，成为了地上世界的一部分。尽管我再也无法回到月之都，可我并不后悔。
这座永远亭中还有一株不可思议的盆栽，其枝干犹如珊瑚一般，枝条上既没有结任何果实，也没有长任何叶片，乍看之下就和枯死了的普通植物毫无二致。
但是我知道这株盆栽也会开出百般娇艳的花朵，结出五彩斑斓的果实，同时也知道如果不满足某些条件的话，这株盆栽的花朵就永远不会绽放。更何况我目前正在尽全力满足这些条件。
#

今天是中秋佳节，绝佳的赏月之夜。依照计划，原本应该召开例行的例月祭，无奈天公不作美，下起了雨，于是我便去永琳所在的房间询问她今晚的安排。
“雨下得这么大，今年的中秋明月怕是看不到了呢。”
我颇为遗憾地说道，事实上却松了口气。说起来，过去我一直畏惧明月高悬的中秋佳节的到来。
不过最近这两三年间，欣赏中秋月色已然成为了一种乐趣，这种变化总是让我感到莫名的恐惧。
尽管深感恐惧，只要看不到满月，那些没来由的杂念就不会在我的脑海中作祟，因此也能松口气。
#

“满月隐藏在云层中，若隐若现的感觉刚刚好呢。”
这阵子，若晴若雨的天气不断周而复始，让人深感人秋后绵长雨季的到来。今年的秋季让人觉得晴天少得可怜，今天自然也不例外，从早开始天空中一直飘着细小的雨点。
“我说永琳，记得告诉因幡她们一声哦?今天的例月祭活动时注意别让身体着凉。”
“哎呀，这份心就不用你操了，辉夜。我早就告诉过她们，如果外面的雨实在是下得太大的话就把活动转移到室内举行。那些兔子们也不想变成落汤鸡之后还在雨中捣团子吧?
“也对，捣的时候掺进雨水的团子谁都不愿意吃呢。”
#

最近，我一直觉得永琳对这群兔子的态度温和了不少。要是在过去，别说是兔子，对于永琳而言地上所有的生物除了是自己的属下之外什么都不是。
即使对于生活在月之都的月之民们而言，兔子也不过是单纯的道具而已，要说这是理所当然的话，的确也是理所当然的。
要说月之民生活的世界和其他生物生活的世界次元不同也毫不为过，他们就是如此高贵的生物。
随后的生活中，不知不觉间永琳开始将我们月之民和地上的兔子们视作同等生物，想必这应该是受了妖怪和人类平等互利、和平共处的幻想乡的影响吧。
#

不过这并不代表我对这样的生活感到厌恶，倒不如说这样的生活比高高在上、受特殊待遇更让我觉得舒畅。
不管怎么说，幻想乡中的月之民只有我和永琳两人，自认高人一等的话只会被孤立，如果地上之民都是道具的话，这道具也未免太多了。
其实我并不想没事翻老黄历，只是因为我从来未曾被视作地上之民的同类。
这和我犯下私自吞服蓬莱之药的大罪，被流放至地上世界时没有任何人将我当成普通人类看待是同一个道理。
说起来被流放至地上时，最初降落的地方和迷失竹林一样，也是一片被迷雾笼罩的竹海。
至今我依然记得当初一脸惊讶地将我捡回去抚养成人的那对老夫妇。
#

他们会觉得惊讶也是理所当然，怎么说那时候我被封入了一株发光的竹子中，身形细小得人类难以想象，被普通人类当做妖怪看待也实属无奈。
当时仍旧是妖怪会掠走人类充当食粮的时代，不知为何那对老夫妇还是把我捡回去当成亲女儿抚养。
我认为当时那对老夫妇刻意将我藏起来的理由，是月之都的监视官会定期地赠与他们财富。月之都的监视官会在同样会发光的竹子中置入黄金，对于老夫妇而言这是作为将我藏匿好的谢礼。
当然，那对老夫妇也会想只要我还在这个家中居住，就会为他们带来源源不断的财富。
或许正因如此，我才受到特殊待遇，决不会被允许迈出家门半步。我觉得他们绝对不想让我被别家的人拐走。
#

不过即使如此，我在地上也显得鹤立鸡群。即使呆在家中无所事事，有关我的传闻也会在世间不胫而走，人们因此聚集到了老夫妇的家中求见我一面。
我尽量避免在地上的公共场合抛头露面，不知不觉间对老夫妇的感恩和不舍之类的复杂情愫也在我心中萌生，老夫妇将我藏匿在家中这件事一直让我心怀感激。
如此生活日复一日地度过，不经意间，地上世界取代了月之都，成为我心中更具魅力的地方。那时我并未对自己施加永远的魔法，尽管影响甚微，我也在被地上世界的污秽逐渐侵蚀着。
#

然而，当时的我仍将自已视作与地上之民截然不同且地位高贵的存在，地上之民在我的眼中等同于道具……幻想乡实在是一片不可思议的土地，妖怪和人类能在此和平共处，过着互不侵犯的生活，陈旧事物与崭新事物在此水乳交融。
在这片土地上出现月之民，融人月之都的最新技术，任何人都不会感到惊讶吧。自诩高人一等只会在这里招来无情的耻笑。
无论如何，幻想乡都是一片让人感到舒畅的大地。若要问这是为何，因为生活在这片土地上，我不必刻意东躲西藏，也不会被他人视作醒目的异类。
#

“最近总是听说雨水的pH值偏低，当然就更不能让她们到外头劳作了。你看，因幡她们毛长得又多厚……”
“呵呵呵，你就放一百二十个心吧。幻想乡的雨水pH值有六呢。如果不是能让草木枯萎的酸雨的话，是不会让她们淋到就掉毛的。”
针对我的玩笑话，永琳居然还一本正经地回答说。，万一她们真的掉毛了，调制治疗脱毛的药物对我而言不过是小菜一碟”。
我也在不断努力学习，试图让自己多少了解地上世界的事。不过，即使如此，我也无法匹敌永琳的勤勉。
?即使用那些我一知半解、?听起来意义不明词汇与她交谈，永琳也能胸有成竹地即刻作答。说老实话，迄今为止我连pH值是个什么玩意儿都还没弄明白……
#

为何她会拥有如此惊人的渊博学识呢。身处月之都时，我并不觉得她的学识如此不可思议，可她不仅上知天文下知地理，连幻想乡和外面世界的一切都了如指掌。
不过，学识渊博者都有个约定俗成的老毛病，他们从来不会将自己的学识以通俗易懂的形式传授给他人。不仅如此，反而会故作高深，以欣赏对方一头雾水的糊涂样为乐。教授和学者其实不过是一，丘之貉，对此我深有感悟。
#

“你说pH值有6 ……也就是说……”
“对于雨水而言这是再正常不过的状态啦，几乎没有任何酸性。”看来pH指的是酸度。还好永琳这次的说明简单易懂。
“难道雨水还会泛酸味吗?无所谓了，就算雨下得再小，也让因幡她们在屋子里捣年糕吧。”
我让永琳把这句话传达给兔子们，永琳点了点头。
#

我离开了永琳那件弥漫着浓烈药味的房间，前往厨房告诉铃仙今天例月祭的相关事项。
“啊，辉夜殿下。我马上就准备好了……”
铃仙已经把雨衣穿戴整齐，一副满心欢喜地要去户外捣年糕的样子。“啊，那个就不必穿了。今天就呆在屋里捣吧。”
“哎?”
铃仙顿时一脸诧异。也难怪，迄今为止，除非屋外交加的风雨让免子们站不稳脚跟，我从来没让免子们在屋里捣过团子。
#

“刚才师傅大人也曾说过，如果雨下得非常大的话就在室内干活，不过现在屋外的雨怎么看都觉得是毛毛雨啊....”
“因为雨水不怎么酸，你们就尽管在屋捣团子吧。”
“因为雨水不怎么酸？这究竟是什么意思……”
“据说pH有六呢，好啦，别事事都创根问底的。只要照我说的去做就？肯定没错。”这原本是永琳的口头禅，毫无缘由地试着将这句话挂在嘴边，那感觉真是相当良好。
“明、明白，多谢您专程跑一趟。那么今天的例月条活动就在永远亭中举行。相应地，届时或许会有些喧闹……”
#

例月祭活动中，兔子们会一边一边歌唱一边捣制年糕和团子，其实这是个和所谓的仪式八竿子打不着的祭典活动。
不过，因为那些免子们都乐在其中，我也就放任自流了。无所谓啦，没准让她们在永远亭中搞活动，她们反而会感到紧张呢。
“真是不可思议啊，明明是神妙的祭典活动，为什么就能吵闹成那副样子呢。”
“是的，啊不是，这回我们会安静地搞活动。”
“免了免了，这也没什么大不了的。我也知道你们在活动中一边放声歌唱一边卖力捣年糕。”
#

“您说我们，其实我是不唱歌的。反正帝她们也从来不会听我的话……”
意识到自己多言的铃仙马上赔礼道“哎呀呀，真是非常抱歉。”紧接着我们便听到了一句口没遮拦的“还没开始吗――？”，猜都不用猜就知道是帝。
“那么，作为吵闹的代价，团子的味道尽量搞得多元化一些吧。”
“这不过是小事一桩。如果您有什么要求的话尽管说，我一定照办。”
我思忖片刻，姑且对铃仙说，“现在已经有三色团子了，不如搞个七彩团子如何？”便离开了厨房。
#

我回到了自己的房间，继续埋头摆弄那株奇妙的盆栽。
即使今天是例月祭，我也是无所事事。不对，不仅是例月祭，平时生活中也没有什么需要我费心的。竹林之外的情报全靠因幡她们传递，偶尔来个急诊病人或是访客，出面应对的职责永琳也会肩承担。什么都不用干的日子听起来挺不错，说实话简直无聊透顶。
记得当初在月之都生活时也是这样，终日无所事事。因为倍感无聊，我开始憧憬地上的多彩生活。
然而来到地上之后，我才第一次领悟到，无所事事与月之都或是地上世界的环境无关，问题出在我自己身上。
将一切都归咎于环境的心灵才是滋生无聊与乏味的温床。
#

因此，作为为了打破终日无所事事的窘境而迈出的第一步，我将摆弄盆栽作为例行日课。仅此而已，我的日常生活中多了一件不得不为之的事，心情也为之焕然一新。
我眼前的这株盆栽，即使我每天都认真端详，也没有任何变化。
但是，地上不存在一成不变之物。有形之物终有一日必定损坏，存活之人终有一日驾鹤西去。存在于地上的万物都无法从这条真理的束缚中逃脱。永琳告诉我，其原因在于污浊之物在地上肆意蔓延。
污秽夺走了物质以及生命的永远，同时为其强加上了寿命这一概念。地上的万物或多或少沾有污秽，因此没有任何事物能永远存在于地上世界，更谈不上什么亘古不变、天长地久。那种事物绝对不可能有。
#

然而，我眼前的这株盆栽并未被地上的污秽侵染，因此才没有任何变化，永远维持现状。尽管乍看之下没有任何成长，可这绝不是因为它已经枯死了。
这是由于直至最近为止，这株盆栽因我的能力而保持着永远的状态。我的能力是“操纵永远与须臾的能力”，是一种创造地上世界中不可能存在，纯洁无暇的永远的能力。
这株盆栽的正体，原本是只在月之都生长的植物“优县华”，与地上世界中传闻三千年才一开花的奇幻鲜花同名。
说这两种植物同名，是由于地上生长着另一种名叫优昙华的植物。某个古老的神话传说称这种花每隔三千年才会绽放一次，事实上不
过是某种难得一见盛开的植物被冠上了这个名称而已。
#

真正的优县华，是一种仅在月之都生长的树木。这种树开花结果之后，如同七色玉石般五彩斑斓的七色果实会挂满枝头。
过去我对某个向我求婚的男人提出的条件中被称作“蓬莱玉枝”的宝物，指的就是这种优昙华开花结果后的枝条。所谓的蓬莱玉枝，就是蓬菜的优昙华开花结果后的产物。
优县华这种植物在月之都生长时，既不长叶片，也不开花，更不结果。尽管卖相看起来干枯孱弱，事实上属于“质朴，致远”这两大日本审美意识中的后者，给人以宁谧致远的印象。
这种树木生长时，看起来既不像在生长也不像已经枯死。不过只要将其枝条带至地面，被地上世界的污秽沾染的枝条的形态就会发生巨大变化。
优昙华会吸取地上的污秽作为养料并不断成长，最终开花结果，使精美绝伦的七色玉石挂满枝头。
#

为何这种吸取污秽为养料的植物也能在一尘不染的月之都生，这实在令人费解，多半这是种在月之都生活的某位贤者创造的人工植物吧。
一旦月之都被污秽侵蚀，月之民们马上便能通过优县华枝头绽放的花朵知晓。
月之使者降临地面时，也会将优县华的枝条随身携带。
随后只要将枝条交给某一位当权者，被污秽彻底侵蚀的当权者会使枝条挂满精美的七色玉石。
当权者手中掌握的实权越大，结出的玉石也就越发精美。当然，权力者们都会将那玉石视作权力的象征。
然而，存在于地上的事物必定会迎来毁灭的一天。
盛者必衰，无论拥有何等力量的人物，有朝一日也会日薄西山，最终粉身碎骨化为尘埃。
届时，优县华的枝条将会成为众人争相抢夺的对象。随后地上世界的平和将会被扰乱，历史的洪流会朝着战火硝烟奔流而去。
#

易言之，优县华是月之民用以在地上世界挑起战乱纷争的植物。
若要问为何要在地上挑起战乱和纷争，只要遍历人类的历史就不难明白。人类的历史，几乎等同于一部战争史，而人类也在大大小小不计其数的战争中获得成长。
没有纷争人类便无从获得成长。自安于现状的那一刻起，人类便放弃了继续生存的执念。月之民的每日生活都必须为地上之民着想，因此称月之民是地上之民的历史的创造者恰如其分。
#

“辉夜，你快看。雨停了云也散了，已经能看到满月了哦。”
背后传来的声响让我睁开了迷离的双眼。刚才我若有所思地端详着盆栽，不知不觉间打起了瞌睡。
“哎呀，真的呢。看着优县华的盆栽发发呆，我都忍不住打了个盹。”
我拥有的这盆优昙华还没有开花，自然更谈不上结出果实。
尽管这是因为我对整个永远亭施与永远，可那永远的魔法已经因为之前的某个骚动被消除了。也就是说，现在的永远亭地处和地上世界相同的历史长河中。
魔法解除后，地上的污秽瞬间遍布整个宅邸，永远亭成为了地上的一部分。这株优昙华的绽放想必也只是时间问题。
#

我和永琳心境的细微变化，恐怕也是受了地上的污秽的影响。虽说被地上的污秽侵染的月之民无法再度回到月之都，我和永琳也不打算再回去。只是可怜了被我们连累，再也无法回到月之都的月兔，铃仙。
说起来，永琳还给铃仙冠上了“优县华”这个名字。
这其中究竟有着怎样的意义呢？难道她觉得铃仙是能吸收在我们身上蔓延的污秽的存在……
这毕竟是永琳的想法，恐怕她是想让不知何为污秽的月兔接触地上的污秽，并期待她结出精美的果实吧。
“哎呀，难道是因为雨停了的关系吗，月亮看起来真漂亮。于是，因幡她们呢？刚才不是告诉过她们在屋里搞例月祭活动么。”
#

永琳笑着回答道:
“例月祭已经移师户外咯。比起憋在屋里，兔子们更喜欢在屋外撒欢呢。
“……你这话是什么意思?”
“不管她们想要干什么，总之魔王不在身边更放得开手脚吧。”
“那索性这样吧，从这次开始，无论屋外下多大的雨，例月祭的活动统一在户外进行。管他屋外是暴雨还是台风呢。”正说着，屋外传来了兔子们爽朗的歌声，看来这群兔崽子们在室内唱得相当憋屈。
#

正当我注视着屋外纵情高歌的兔子们时，永琳注意到了优昙华的盆栽。
“……看起来还是没有任何变化呢，不过马上就要开始成长了吧。等到它的花朵默默地绽放之后，应该就能结出漂亮的七彩玉石了。真让人期待啊。”
“是啊，谁让七彩缤纷的优昙华树是地上之民的特权呢。说起来，今天我能先人一步享受那七色果实的美味呢。”
“这是怎么回事？”永琳歪起了脑袋。世间无论何事，只要能让永琳抱有疑问，我就会觉得尤其欣慰。
#

“我对铃仙提议说今晚不弄三色的团子，而是改做七色的。”
“原来如此，这听起来挺有趣的。不过，如果那团子是像优县华那样的七色的话，我就有些不想吃了。”
“为什么?”
“你看啊，蓬莱之玉中有青色和蓝色的吧，那种颜色的团子你看了还有食欲吗？”
#

窗外传来的兔子们的歌声的音量突然间猛增，卖力捣年糕的声音与和太鼓的敲击声交织在一起，奏响了一曲激昂的旋律。
兔子们拥有一种不可思议的能力，能够以心传心，兔族同胞之间无需过多的言语维系，也能保证坚不可摧的协作性。
铃仙能与远在月之都中的月兔们沟通，帝她们不用喊口令号子也能合着节拍一起载歌载舞。只要对各自捣年糕的时机和周期分别加以协调，就能演奏出不可思议的乐曲声。
#

我和永琳在起居室里欣赏兔子们演奏的音乐，品味杯中的香茶，享受漫漫秋夜为我们带来的丝丝清凉。中秋夜明月什么的对于我而言其实无所谓，我也就不在户外乘风凉晒月亮了。
“说起来，今天兔子们捣鼓出的旋律真是惊人啊。打了鸡血还是吃了兴奋剂吗?
“因为天上有中秋明月啊。”
这么说着永琳还补上了一句“就好像狂热的凯卡克舞一般呢”，她在说什么我完全摸不着头脑。
#

“说起来，永琳，虽说都已经是两个月以前的事了....我总觉得从那之后就不太安稳?
“是啊，的确有人在大肆传播，现在各地都在谈论和月球相关的话题。”
没错。两个月前，月之都中出现了以永琳为首的叛乱组织预谋攻占月之都的传闻，某只月免更是逃亡至地上，躲进了博丽神社。一系列不可思议的事件轮番上演。
在此之前，传闻月之都中开始显露内乱的征兆，某些大胆妄徒预谋在月之都中掀起暴动，这是毋庸置疑的事实。
事件的主谋为了找替罪羊洗脱嫌疑，使用了永琳的名号。不，倒不如说永琳的名号自然而然地被月之都划入了黑名单。
#

关于这一系列事件，永琳向月之都中对自己抱有绝对信任的人物送去了亲笔题写的书信，我想事情应该能就此解决，也不需要再担心什么，可不知为何幻想乡中有关月球的流言却不绝于耳，更有愈演愈烈之势。
“知道两个月前出现在神社的妖兔就是月兔的人应该不可能有才对。”
“恐怕是中间出了出乎我们意料之外的岔子。”
#

“岔子?”
“当时我们并未把真正的幕后黑手考虑在内。”
“……幕后黑手什么的真的存在吗?而且这个幕后黑手在谋划什么我也是一头雾水。不过，如果这一切并非意外，?而是幕后黑手预先设计好的会怎样?”
“如果有这种可能性的……我只能说对方真有一手。”
永琳无奈地摊开手，耸了耸肩，“现在我已经没有任何办法联络身处月之都的她们俩了。”
#

问题不仅仅在于月面战争。传闻吸血鬼貌似正在组装登月火箭，神社和香霖堂的主人还将幻想乡的各个角落都拖下了水，试图制造大规模装置。
“难道说那只吸血鬼是幕后黑手？据说她正在开发登月火箭，现在都快建造完成了，而且就能力而言，吸血鬼是最有实力进攻月之都的。”
“那种可能性的确很高，不过如果真是这样，我会成为替罪羊，月面旗帜被拔掉，月兔降临地面，这一系列事件就全都是偶然了。”
“这是为什么？”尽管永琳的答案我能想象得到，不过姑且还是一问。无奈最终得到的果然是“那种小孩子可没这么好的头脑”这般在意料之中的回答，不禁让我觉得有些遗憾。
#

太鼓激烈的敲击声取代了兔子们嘈杂的喧嚣，这个月的例月祭也顺利结束了。
我和永琳为了一览例月祭结束的瞬间而来到了室外，团子已经被整齐地堆放在一个巨大的盘子中。团子的确被精心制作成了七种颜色，不过看那几近把人晃瞎的红，以及色彩过度艳丽的青，七种组合在一?起仿佛能让人看到幻觉的色彩实在无法勾起观者一点的食欲。
“啊，辉夜殿下，师傅大人。现在我们正在收拾，请稍等片刻。铃仙命令帝去收拾捣团子用的石臼。我仰望夜空，四处搜寻被称作中秋夜明月的月亮，然而空中已然不见满月的踪影。
#

“不是说雨停了吗……又躲进云层了吧。”
“嗯嗯，就是这么一回事，能看到月亮的时间只有那么一小段。难得雨停了能来户外赏月……您怎么了？”
“没什么，只是觉得你身为从月之都逃出来的月兔，却说‘难得能出来赏月’，这实在有些好笑。”我淡淡地笑了。
铃仙有些不好意思地说道：“好歹我也在地上生活了这么久，怎么都该人乡随俗，成为地上之民的一份子了嘛。”
#

地上的兔子们此刻都在老老实实地收拾石白。要是换作平时，这群兔子收拾了一半就不知道溜去哪儿玩了，对此铃仙也感到不可思议。
其实这也是理所当然，因为今天的团子中没有掺杂过去经常下的那种药。
一直以来永琳都以兔子们会偷吃为前提，在团子中掺人能让兔子们亢奋的药物，以此炒热祭典活动的气氛。
不过今天的团子我们也要吃，所以永琳就把兴奋药换成了强身健体的滋补药。不怎么亢奋的兔子们比平时顺从得多。
即使如此，兔子们依旧将祭典活动的气氛渲染得空前高涨。无关药物也能造就如此热火朝天的气氛，那自然比什么都好。
#

“说起来，铃仙，记得我说过让你做七色的团子……你做的那些七色团子可真够鲜艳的。”
未了，我还颇为讥讽地补了一句“桃色、白色和艾草色做成的三色团子可真让人有食欲啊。”
“需要我进行说明吗？这次的团子我做成了红、橙、黄、绿、青、蓝、紫七种颜色。每一种颜色都让我花了不少心思去寻找素材……不想只考虑了如何做出颜色。虽说内容听了也不见得能勾起您的食欲，需要我说明吗?”
“不，免了。”听之前我就没多少食欲，听了之后估计食欲还会大减，如此一来今晚非郁闷死我不可。
#

永琳捏起一个青色的团子，放在嘴里咀嚼着，边嚼边咕哝“没想到这还挺好吃的”。原来如此，正是因为这份敢于挑战任何未知事物的求知欲才让永琳如此博学，我真是佩服至极。
“说起来我得换一个话题，二位是否感受到从上上个月起有一股不安定的波动从月球传到了地上?”
我和永琳彼此对视一眼，?异口同声地说道:
“这事犯不着你这兔子操心。”
#

铃仙的表情显得有些不安。
“不，或许这并不是什么大不了的事，之前吸血鬼身边的女仆登门造访，向我询问永远亭中是否有登月途径的相关资料。”
“哎呀，我们家也来过了啊。
“您这‘我们家也来过了’指的是?”
“刚才我和永琳两个人一直在聊天，谈到吸血鬼正在制造登月的火箭。据说为了制造火箭，红魔馆的家伙们正在四处网罗资料呢。”
#

“原来是这样。不过那时候我即刻请她回去了……”
“为什么你要急着把人家赶回去?”
“哎？可是您看啊，我们没有理由协助地上的妖怪们进犯月球吧。这种事不需要辉夜殿下和师傅大人操心。”
“如果铃仙没把她赶回去的话，我还能招待她喝杯茶呢。虽然资料我不会拱手相送。”
#

或许因为空中缺少了雨点们翩翩起舞的身姿，冰凉的清风顺着门缝窗隙悄悄摸进了屋子。这个月的满月之夜看似也能安然无恙地度过。
说句老实话，现在我已经不会再想月之都的使者是否会在满月之夜前来迎接我。这种事不用想都知道已经不可能了。
若要问这是为什么，上千年间我一直胆颤心惊地生活在这间被施加了永远的魔法的大屋中，然而大约三年前，魔法被解除，我的心境也即刻发生了转变。
#

如果这也是由于地上蔓延的污秽的影响，地上之民这一群体将会何等善变，生活又将何等轻松愉悦。
永远不必为了同件事伤透脑筋，不悦的回忆会逐一从记忆中褪色消失。既然我已经发生如此这般的转变，永琳必定也和我一样吧?
永琳在世上生活的年月比我这样的小丫头久远得多。
她在月之都生活的时间也相当长，并且身处月之贤者这一重要立场。当初我们降临地上时，她持远离人类社会、过隐居生活的意见。
#

正因如此，即使我停止对永远亭施加永远的魔法，我也觉得永琳绝对不会发生任何转变。不知为何，我总觉得永琳与地上世界的污秽无缘。当然，事实并不像我想象的那样，永琳的言行举止也在匆匆流逝的日常生活中逐渐显露变化。
永琳在地上开了家药铺，自己则出任医师，现在已经是一代名医。但凡人类村落中的不治之症，只要去永远亭走二趟保管药到病除。这事放在过去的永琳身上简直让人无法想象。过去只会将地上之民看作奴仆看待的人物，现今已经成为在地上救死扶伤的人民公仆。
#

至于开办药铺的理由，永琳似乎曾说过，“?既然从今以后我将作为地上之民而活，那么必须履行地上之民应尽的义务。劳动中人人为我，我为人人，这就是地上之民应尽的义务。”换句话说，在地上世界里天上不会掉馅饼，不劳者无食。
这些话我多少都能理解。将我藏起来的那对老夫妇也是如此，被月之都强行赐予的金银财宝夺去了原本平和安详的生活。地上之民不得期待自己劳动成果以上的回报，因为那必然会招致不幸。
#

不过，即便理解这一点，我依然未曾将其付诸实践。不止我一个，印象中幻想乡中有不少这样未能尽到地上之民职责的人物。
我将心中的烦恼向永琳倾诉，却被她用一句“辉夜只管做自己想做的事就好。如果没有什么特别想做的，那就把寻找自已想做的事当成眼下的工作”给打发了。
迄今为止，我依然还未找到身为地上之民的自己应做的事。不过我想待到优昙华花朵绽放之时，某些事就应该会悄然上演。
倒不如说，等我明白什么才是自己该做的事时，没准优县华就会绽放它那难得一见的花朵。
#

“铃仙，既然吸血鬼的使者来了，你让她直接到我这里来不就好了吗”
永琳忠告铃仙的语调有些严苛，铃仙马上慌慌张张地回答道：
“师傅大人，非常抱歉，难道说您想从那位女仆口中套出相关的情报吗？如果真是这样的话，不妨再把她找来……”
“当面见到她的话，我就能郑重其事地拒绝了。”永琳笑着说道。铃?仙也颇为安心地俏皮道：“那?今后打发人的重任就交给师傅您了。”
#

“反正我们再也无法回到月球了。”
“是啊……就连我们手头也没有返回月球的方法，吸血鬼到底想怎样登月呢。”铃仙面朝原本明月高悬的夜空喃喃自语。
永琳注视着铃仙表情的细微变化，仿佛为了让她安心般说道:
“所以呢，铃仙你根本不必担心地上这股不稳定的波动。”
#

铃仙像是要驱散心中的疑虑般，转头面向自己的师傅，为了再度得到确认而提问道：
“也就是说，地上没有任何人能前往月球，我们也就能放心了？听到她这番话，不知为何永琳开始放声大笑，随后又马上回答道：“啊哈哈，不对不对，我们已经不再是月之民，而是老老实实窝在地上生活的人类和妖怪，何苦还为月之都操那份心呢。”
这番话也让我不禁哈哈大笑。与此同时，我决定将今后发生的一系列不可思议事件全权交由永琳处理。
